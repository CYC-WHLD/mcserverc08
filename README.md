## 📖 概覽 (Overview)

**C08 Minecraft Server** 是一個結合了尖端科技感與極致遊戲體驗的社群伺服器。我們的標誌靈感來自於現代 AI 的幾何結構，象徵著網路、宇宙與人與人之間的深度連結。

在 C08，我們提供了一個穩定且充滿創意空間的環境，讓玩家可以在廣闊的開放世界中建築、探險，並與全球的玩家共同協作。
</div>

<div style="background-color:#fdf6e3; border-radius:10px; padding:20px; margin-bottom:20px;">
## ✨ 伺服器特色 (Highlights)

* **極簡科技風格**：擁有專屬的現代幾何標誌，為伺服器注入未來感。
* **開放世界與生存**：豐富的地形生成與平衡的生存機制。

## 🚀 快速加入 (Quick Start)

### 伺服器資訊
* **伺服器地址 (IP):** `play.c08.net`
* **版本:** Minecraft Java Edition 1.20.11
* **正版驗證:** 開啟 (Online Mode)

### 加入步驟
1. 啟動 Minecraft 遊戲啟動器。
2. 選擇 **多人遊戲 (Multiplayer)** -> **新增伺服器 (Add Server)**。
3. 輸入伺服器名稱 `C08 Server` 及 IP。
4. 點擊 **完成** 並加入，開始您的冒險！
</div>

技術架構 (Project Structure)

* `/plugins`: 自定義插件與配置，優化遊戲性能。
* `/scripts`: 自動備份與重啟腳本。
* `/assets`: 伺服器專屬資源包 (Resource Pack) 與 logo 素材。
* `server.properties`: 經過深度優化的伺服器參數。
